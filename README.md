# deLoupe

deLoupe is a tool to extract `.cloupe` files generated by cellranger/spaceranger.

## Installation

Download the precompiled binary for your platform from the [releases page](https://github.com/ShuyangXu/deLoupe-dist/releases).

## Usage

### Basic Commands

Extract a cloupe file:
```bash
deloupe extract <input.cloupe> (-o <output_dir>)
```

View information about a cloupe file:
```bash
deloupe info <input.cloupe>
```

### Platform-Specific Instructions

#### Linux

```bash
deloupe extract cloupe.cloupe -o outdir/
```

#### macOS

##### arm64

**Note:** Files downloaded via browser may have the `com.apple.quarantine` extended attribute applied by macOS. Since the binary is ad-hoc signed, you may need to remove this attribute before running:

```bash
# xattr -d com.apple.quarantine deloupe
```

Then run:
```bash
deloupe extract cloupe.cloupe -o outdir/
```

##### amd64

Intel-based macOS builds are not available. GitHub Actions has deprecated Intel macOS runners and reduced resource allocation for legacy architecture support.

#### Windows

```bash
deloupe.exe extract cloupe.cloupe -o outdir\
```

# cellranger testrun --id TESTRUN

# Extract the cloupe file
deloupe extract TESTRUN/outs/cloupe.cloupe -o test/
